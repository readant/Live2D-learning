# Live2D 桌宠项目概述

## 项目简介

本项目是一个基于 **PyQt5 + Live2D** 实现的桌面宠物程序，实现了以下核心功能：

- 🖱️ **鼠标跟随**：头部和眼球跟随鼠标移动
- 💨 **呼吸动画**：随机呼吸微动效果
- 👋 **点击交互**：点击桌宠触发挥手动作
- 😊 **表情切换**：支持开心/认真/犯困三种表情
- 🎨 **无框窗口**：无标题栏、可拖动、始终置顶

## 技术栈

| 技术 | 版本要求 | 说明 |
|------|---------|------|
| Python | 3.8+ | 建议 3.10+ |
| PyQt5 | >= 5.15.0 | GUI 框架 |
| live2d-py | >= 0.6.1 | Live2D SDK Python 封装 |
| PyOpenGL | >= 3.1.0 | OpenGL 支持 |
| numpy | >= 1.20.0 | 数值计算 |

## 模型文件说明

本项目使用的模型文件为 **Hiyori** (日野雏)，这是一个免费的 Live2D 示例模型。

模型文件来源：`e:\Projects0\Trae\Live2D-learning\resources\models\hiyori_free_t08.model3.json`

### 模型包含的内容

- `.model3.json` - 模型配置文件
- `.moc3` - 模型数据文件
- `.physics3.json` - 物理演算配置
- `texture_00.png` - 纹理图集
- `motion/*.motion3.json` - 动作文件

### 动作组

| 动作组名 | 说明 | 触发条件 |
|---------|------|---------|
| Idle | 待机动画 | 默认循环播放 |
| Tap | 点击触发 | 点击模型 |
| Flick | 抚摸动画 | 快速滑动 |
| FlickDown | 下拉动画 | 向下拉动 |

## 功能模块划分

### 1. 配置模块 (config/)

集中管理所有配置参数，便于修改和维护。

### 2. Live2D 模块 (src/live2d/)

封装 Live2D 模型操作的核心接口。

### 3. 核心功能模块 (src/core/)

实现鼠标追踪和动画控制逻辑。

### 4. UI 模块 (src/ui/)

PyQt5 窗口组件和事件处理。

## 运行方式

```bash
# 安装依赖
pip install -r requirements.txt

# 运行程序
python main.py
```

## 快捷操作

- **左键点击**：触发挥手动作 + 切换表情
- **右键菜单**：手动选择表情或退出
- **拖拽窗口**：按住左键拖动

## 扩展建议

1. **添加新模型**：将新模型文件放入 `resources/models/` 目录
2. **修改窗口大小**：编辑 `config/settings.py` 中的 `WINDOW_WIDTH` 和 `WINDOW_HEIGHT`
3. **添加新表情**：在 `src/live2d/expression.py` 中添加新的表情配置
4. **集成到其他项目**：参考 `src/live2d/model.py` 中的 `Live2DModelInterface` 类
