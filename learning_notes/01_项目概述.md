# Live2D 技术学习项目概述

## 项目定位

> **核心目标**：深入理解 Live2D 技术原理，掌握其核心能力，并探索多样化的应用场景。
> 
> **桌宠只是起点**：桌面宠物是学习过程中的一个实践载体，而非唯一目标。我们的重点是掌握 Live2D 的**核心技术**和**扩展能力**。

---

## 技术学习目标

本项目旨在掌握以下核心技术能力：

| 技术领域 | 核心能力 | 应用场景 |
|----------|----------|----------|
| **模型渲染** | 2D 网格变形、骨骼绑定、纹理映射 | 虚拟角色展示 |
| **动画系统** | 关键帧动画、曲线插值、动画混合 | 角色动作表现 |
| **参数系统** | 参数映射、表情控制、实时驱动 | 面部捕捉、交互反馈 |
| **物理引擎** | 弹簧阻尼、自然运动、物理模拟 | 头发/衣服飘动 |
| **面部捕捉** | MediaPipe 集成、姿态估计、平滑滤波 | 实时面部驱动 |
| **动作录制** | motion3.json 格式、动作导出 | 动作复用、跨平台 |

## 应用场景发散

Live2D 技术具有广泛的应用场景，远远超出桌宠范畴：

### 🎮 游戏领域
- **角色立绘**：AVG 游戏角色展示
- **虚拟 NPC**：游戏内交互式角色
- **表情系统**：丰富的角色表情表现

### 🎤 虚拟主播
- **实时直播**：面部捕捉驱动虚拟形象
- **短视频创作**：虚拟角色内容生成
- **直播互动**：观众互动反馈

### 🏫 教育领域
- **教育机器人**：交互式学习助手
- **虚拟讲师**：在线课程虚拟形象
- **知识科普**：趣味化知识传播

### 💬 社交娱乐
- **聊天机器人**：个性化虚拟形象
- **表情包生成**：动态表情创作
- **虚拟陪伴**：情感陪伴角色

### 🖥️ 企业应用
- **虚拟客服**：智能客服形象
- **产品展示**：交互式产品介绍
- **品牌形象**：企业虚拟代言人

### 🎨 创意工具
- **动画制作**：低成本 2D 动画
- **角色设计**：快速原型开发
- **AR/VR 集成**：虚实融合体验

## 技术栈

| 技术 | 版本要求 | 说明 |
|------|---------|------|
| Python | 3.10+ | 主开发语言 |
| PyQt5 | >= 5.15.0 | GUI 框架 |
| live2d-py | >= 0.6.1 | Live2D SDK Python 封装 |
| PyOpenGL | >= 3.1.0 | OpenGL 渲染支持 |
| numpy | >= 1.20.0 | 数值计算 |
| opencv-python | >= 4.5.0 | 计算机视觉 |
| mediapipe | >= 0.8.0 | 面部捕捉 |

## 总结

本项目以**桌宠为实践载体**，但核心目标是**掌握 Live2D 技术体系**。通过系统化的学习路径，您将获得：

> **记住**：桌宠是起点，而非终点。掌握核心技术后，您可以将 Live2D 应用到任何需要虚拟角色的场景中！
