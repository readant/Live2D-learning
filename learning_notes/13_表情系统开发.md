## 表情系统概述

### 表情的作用

表情是角色传达情感的重要方式，增强角色的表现力和亲和力：

| 表情类型 | 用途 | 触发场景 |
|----------|------|----------|
| 开心 | 正面反馈 | 被点击、互动成功 |
| 生气 | 负面情绪 | 长时间无操作 |
| 惊讶 | 强烈反应 | 突然事件 |
| 犯困 | 待机状态 | 长时间无操作 |
| 思考 | 等待输入 | 提问等待回答 |

### 表情与动作的关系

```
动作（Action）          表情（Expression）
    │                        │
    ▼                        ▼
身体运动              面部参数变化
（站立、坐下）         （眉、眼、嘴）
    │
    ▼
动作完成后可切换表情
```

## 表情参数配置

### 常用表情参数

| 参数名 | 作用 | 取值范围 | 说明 |
|--------|------|----------|------|
| ParamEyeLOpen | 左眼睁开度 | 0 ~ 1 | 0=闭眼，1=完全睁开 |
| ParamEyeROpen | 右眼睁开度 | 0 ~ 1 | 同上 |
| ParamEyeLSmile | 左眼微笑度 | 0 ~ 1 | 0=正常，1=笑眼 |
| ParamEyeRSmile | 右眼微笑度 | 0 ~ 1 | 同上 |
| ParamEyeBallX | 眼球横向 | -1 ~ 1 | -1=左，1=右 |
| ParamEyeBallY | 眼球纵向 | -1 ~ 1 | -1=下，1=上 |
| ParamBrowLAngle | 左眉角度 | -1 ~ 1 | 皱眉/抬眉 |
| ParamBrowRAngle | 右眉角度 | -1 ~ 1 | 同上 |
| ParamBrowLForm | 左眉形状 | -1 ~ 1 | 弯曲程度 |
| ParamBrowRForm | 右眉形状 | -1 ~ 1 | 同上 |
| ParamMouthForm | 嘴巴形状 | -1 ~ 1 | -1=不开心，1=开心 |
| ParamMouthOpenY | 嘴巴张开度 | 0 ~ 1 | 说话/张嘴 |
| ParamCheek | 脸颊红晕 | 0 ~ 1 | 害羞/开心 |

### Blend 类型

| 类型 | 说明 | 用途 |
|------|------|------|
| Add | 增量叠加 | 在当前值基础上增加 |
| Multiply | 乘法叠加 | 与当前值相乘 |
| Override | 完全覆盖 | 直接替换当前值 |

## 表情切换逻辑

### 表情切换流程

```
用户触发
    │
    ▼
检查冷却时间 ──是──→ 忽略切换
    │
    否
    │
    ▼
获取目标表情配置
    │
    ▼
淡入淡出过渡
    │
    ▼
应用表情参数
    │
    ▼
启动冷却计时器
```

### 表情切换实现

```python
class ExpressionController:
    def __init__(self):
        self.current_expression = Expression.SERIOUS
        self.target_expression = None
        self.transition_time = 0.3  # 过渡时间（秒）
        self.transition_progress = 1.0
        self.cooldown = 0.5  # 切换冷却
        self.cooldown_timer = 0.0

    def switch_expression(self, new_expression):
        # 检查冷却
        if self.cooldown_timer > 0:
            return False

        # 设置目标表情
        self.target_expression = new_expression
        self.transition_progress = 0.0
        self.cooldown_timer = self.cooldown
        return True

    def update(self, delta_time):
        # 更新冷却
        if self.cooldown_timer > 0:
            self.cooldown_timer -= delta_time

        # 更新过渡
        if self.transition_progress < 1.0:
            self.transition_progress += delta_time / self.transition_time
            if self.transition_progress >= 1.0:
                self.transition_progress = 1.0
                self.current_expression = self.target_expression
                self.target_expression = None

        # 应用表情参数
        self.apply_expression()

    def apply_expression(self):
        if self.target_expression:
            # 过渡中的插值计算
            t = self.ease_in_out(self.transition_progress)
            config = self.mix_expressions(
                self.current_expression,
                self.target_expression,
                t
            )
        else:
            config = EXPRESSION_CONFIGS[self.current_expression]

        # 应用到模型
        for param_id, value in config.items():
            self.model.set_parameter(param_id, value)

    @staticmethod
    def ease_in_out(t):
        return t * t * (3 - 2 * t)  # 平滑插值
```

### 随机表情切换

```python
import random

class RandomExpression:
    def __init__(self):
        self.expressions = [Expression.HAPPY, Expression.SERIOUS, Expression.SLEEPING]
        self.switch_interval = 10.0  # 平均10秒切换
        self.timer = random.uniform(5.0, 15.0)

    def update(self, delta_time):
        self.timer -= delta_time
        if self.timer <= 0:
            # 随机选择表情（避免重复）
            new_expr = random.choice([e for e in self.expressions if e != self.current])
            self.expression_controller.switch_expression(new_expr)
            self.timer = random.uniform(5.0, 15.0)
```

## 复合表情

### 复合表情原理

复合表情是将多个基础表情按比例混合：

```python
class CompoundExpression:
    def __init__(self):
        self.base = EXPRESSION_CONFIGS[Expression.SERIOUS].copy()
        self.modifiers = []

    def add_modifier(self, modifier_name, intensity):
        """添加表情修饰器

        Args:
            modifier_name: 修饰器名称（如 "smile", "surprise"）
            intensity: 强度 0.0 ~ 1.0
        """
        modifier = EXPRESSION_MODIFIERS[modifier_name]
        self.modifiers.append((modifier, intensity))

    def evaluate(self):
        """计算最终的复合表情参数"""
        result = self.base.copy()

        for modifier, intensity in self.modifiers:
            for param_id, value in modifier.items():
                if param_id in result:
                    # 增量混合
                    result[param_id] += value * intensity
                else:
                    result[param_id] = value * intensity

        return result

# 表情修饰器定义
EXPRESSION_MODIFIERS = {
    "smile": {
        "ParamEyeLSmile": 0.5,
        "ParamEyeRSmile": 0.5,
        "ParamMouthForm": 0.3,
    },
    "surprise": {
        "ParamEyeLOpen": 0.5,
        "ParamEyeROpen": 0.5,
        "ParamMouthOpenY": 0.3,
    },
    "anger": {
        "ParamBrowLAngle": -0.5,
        "ParamBrowRAngle": -0.5,
        "ParamMouthForm": -0.3,
    }
}
```

### 使用示例

```python
# 创建复合表情：60%开心 + 40%惊讶
compound = CompoundExpression()
compound.add_modifier("smile", 0.6)
compound.add_modifier("surprise", 0.4)

final_expression = compound.evaluate()
```

## 项目中的表情实现

### 表情应用逻辑

项目中的表情实现在 `src/live2d/expression.py`：

```python
def apply_expression(model, expression_type):
    """将指定表情应用到模型

    Args:
        model: Live2D 模型实例
        expression_type: Expression 枚举值
    """
    config = EXPRESSION_CONFIGS.get(expression_type)
    if not config:
        return

    for param_id, value in config.items():
        model.set_parameter(param_id, value)
```

### 表情切换时机

```python
class DesktopPetWidget:
    def on_click(self):
        # 点击时切换表情
        expressions = [Expression.HAPPY, Expression.SERIOUS]
        current_idx = expressions.index(self.current_expression)
        next_idx = (current_idx + 1) % len(expressions)
        self.expression_controller.switch_expression(expressions[next_idx])

    def on_idle_timeout(self):
        # 待机超时切换到睡眠表情
        self.expression_controller.switch_expression(Expression.SLEEPING)
```

## 表情系统设计指南

### 设计原则

| 原则 | 说明 |
|------|------|
| 简洁性 | 每个表情应有明确的情感特征 |
| 可区分性 | 不同表情之间应有明显区别 |
| 自然过渡 | 表情切换应平滑自然 |
| 适度数量 | 表情数量适中，避免过于复杂 |

### 表情数量建议

| 应用场景 | 推荐数量 | 示例 |
|----------|----------|------|
| 简单桌宠 | 3-5 个 | 开心、认真、犯困、生气 |
| 中等复杂度 | 6-8 个 | 添加惊讶、害羞、思考 |
| 高级应用 | 10+ 个 | 细分各种情绪变化 |

### 表情调试技巧

```python
def debug_expression(model, expression_type):
```
