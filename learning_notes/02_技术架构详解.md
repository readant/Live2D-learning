## 窗口系统架构

### QOpenGLWidget 集成

本项目使用 `QOpenGLWidget` 作为 Live2D 模型的渲染容器，这是最关键的技术决策。

#### 为什么使用 QOpenGLWidget？

1. **硬件加速**：利用 GPU 进行渲染，性能更好
2. **官方支持**：live2d-py 官方推荐在 OpenGL 上下文中渲染
3. **上下文共享**：通过 `AA_ShareOpenGLContexts` 属性支持多窗口

#### 三个核心方法

```python
class DesktopPetWidget(QOpenGLWidget):
    
    def initializeGL(self):
        """初始化 OpenGL 上下文和 Live2D 模型"""
        self.makeCurrent()  # 获取当前 OpenGL 上下文
        live2d.init()
        live2d.glInit()
        self.live2d_model = live2d.LAppModel()
        self.live2d_model.LoadModelJson(model_path)
    
    def paintGL(self):
        """渲染循环，每帧调用"""
        live2d.clearBuffer()  # 清除缓冲区
        self.live2d_model.Update()  # 更新模型状态
        self.live2d_model.Draw()  # 绘制模型
    
    def resizeGL(self, w, h):
        """窗口大小改变时调用"""
        self.live2d_model.Resize(w, h)  # 调整模型大小
```

## 模块详解

### 1. 鼠标追踪模块 (MouseTracker)

**位置**：`src/core/mouse_tracker.py`

**功能**：追踪鼠标位置，计算头部和眼球转动角度

**核心算法**：
- 将鼠标位置转换为相对于模型中心的偏移量
- 使用反正切函数计算角度
- 应用角度限制，防止过度转动

```python
# 视线角度计算公式
angle_x = atan2(mouse_x - center_x, distance) * factor
angle_y = atan2(mouse_y - center_y, distance) * factor
```

**关键参数**：
- `gaze_center`: 视线中心点
- `max_gaze_angle`: 最大注视角度 (30度)
- `gaze_smoothing`: 注视平滑系数 (0.15)

### 2. 动画控制模块 (AnimationController)

**位置**：`src/core/animation_controller.py`

**功能**：管理呼吸动画和挥手动作

**呼吸动画**：
- 使用正弦波实现平滑的上下浮动
- 周期：3秒完成一次呼吸
- 振幅：5像素

**挥手动作**：
- 触发间隔：2秒
- 检测点击区域，播放对应动作

### 3. Live2D 模型接口 (Live2DModelInterface)

**位置**：`src/live2d/model.py`

**功能**：封装 live2d-py 库的操作，提供统一的接口

**关键方法**：
- `load()`: 加载模型
- `set_parameter()`: 设置模型参数
- `start_motion()`: 播放动作
- `hit_test()`: 点击测试

### 4. 渲染器模块 (Live2DRenderer)

**位置**：`src/live2d/renderer.py`

**功能**：负责将模型绘制到窗口

**注意**：当前版本中，真实渲染由 `paintGL()` 直接调用 Live2D 模型，渲染器主要用于模拟模式。

## 渲染模式切换

本项目支持两种渲染模式：

### 真实渲染模式
- 使用 `QOpenGLWidget`
- 调用 `live2d.LAppModel` 的 `Update()` 和 `Draw()`
- 渲染真实的 Live2D 模型

### 模拟渲染模式
- 当 OpenGL 初始化失败时自动切换
- 使用 `QPainter` 绘制简单的圆形
- 用于调试和演示

```python
def paintGL(self):
    if self.live2d_model:
        # 真实渲染
        live2d.clearBuffer()
        self.live2d_model.Update()
        self.live2d_model.Draw()
    else:
        # 模拟渲染
        painter = QPainter(self)
        self._fallback_render(painter)
```

## 性能优化建议

1. **使用共享 OpenGL 上下文**
   ```python
   QApplication.setAttribute(Qt.AA_ShareOpenGLContexts, True)
   ```

2. **合理的帧率**
   - Live2D 模型：30 FPS 足够
   - 避免过高帧率造成性能浪费

3. **避免频繁的 OpenGL 上下文切换**
   - 使用 `makeCurrent()` 确保在正确的上下文中操作

4. **纹理缓存**
   - Live2D SDK 会自动缓存纹理
   - 避免重复加载相同的纹理
