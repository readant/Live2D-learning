### 练习 1: 添加新表情

**难度**: ⭐⭐☆☆☆

**目标**: 学习如何为 Live2D 模型添加自定义表情。

**任务**:
1. 添加一个新的表情 "惊讶" (SURPRISED)
2. 为这个表情定义合适的参数值
3. 在右键菜单中添加切换选项

**关键参数**:
- `ParamEyeLOpen/ParamEyeROpen`: 眼睛睁开程度
- `ParamMouthOpenY`: 嘴巴张开程度
- `ParamBrowLForm/ParamBrowRForm`: 眉毛形状

**答案要点**:
```python
Expression.SURPRISED: {
    "ParamEyeLOpen": 1.0,
    "ParamEyeROpen": 1.0,
    "ParamMouthForm": 0.0,
    "ParamMouthOpenY": 0.8,
    "ParamBrowLForm": 0.8,
    "ParamBrowRForm": 0.8,
}
```

### 练习 2: 修改鼠标跟随灵敏度

**难度**: ⭐☆☆☆☆

**目标**: 学习如何调整 Live2D 模型的鼠标跟随效果。

**任务**:
1. 修改 `max_gaze_angle`（当前 30°）
2. 调整 `max_distance`（当前 400）
3. 观察不同数值的效果

**参数说明**:
| 参数 | 作用 | 推荐范围 |
|------|------|----------|
| `max_gaze_angle` | 最大注视角度 | 20-50 |
| `max_distance` | 最大有效距离 | 200-500 |

**实验建议**:
- 尝试 `max_gaze_angle = 45, max_distance = 300`（高灵敏度）
- 尝试 `max_gaze_angle = 20, max_distance = 500`（低灵敏度）

### 练习 3: 添加新动作

**难度**: ⭐⭐⭐☆☆

**目标**: 学习如何触发 Live2D 模型的不同动作。

**任务**:
1. 使用 `StartRandomMotion` 随机播放动作
2. 使用 `StartMotion` 播放指定动作
3. 了解不同的动作组（Tap, Idle, Flick）

**API 说明**:
```python
# 随机播放动作组
self.live2d_model.StartRandomMotion("Tap", priority)

# 播放指定动作
self.live2d_model.StartMotion("Tap", 0, priority)
```

**常见动作组**:
- `Tap`: 点击触发
- `Idle`: 待机动作
- `Flick`: 滑动触发

### 练习 4: 实现拖拽移动

**难度**: ⭐⭐⭐⭐☆

**目标**: 学习如何让桌宠窗口可以被拖拽移动。

**任务**:
1. 实现窗口拖拽功能
2. 区分拖拽和点击事件
3. 添加边界限制（可选）

**实现思路**:
```python
# 1. 鼠标按下：记录初始位置
def mousePressEvent(self, event):
    self._is_dragging = True
    self._drag_start_pos = event.pos()
    self._window_start_pos = self.pos()

# 2. 鼠标移动：更新窗口位置
def mouseMoveEvent(self, event):
    if self._is_dragging:
        delta = event.pos() - self._drag_start_pos
        self.move(self._window_start_pos + delta)

# 3. 鼠标释放：检测是否为点击
def mouseReleaseEvent(self, event):
    distance = (event.pos() - self._drag_start_pos).manhattanLength()
    if distance < self._click_threshold:
        # 处理点击事件
    self._is_dragging = False
```

**点击阈值**:
通常使用 5-10 像素作为点击/拖拽的判断边界。

## 如何使用练习

1. **先尝试自己完成**
   - 阅读练习要求
   - 查看提示
   - 独立编写代码

2. **运行你的代码**
   - 确保代码能正常运行
   - 观察效果

3. **对比答案**
   - 查看答案目录中的实现

4. **扩展练习**
   - 在完成基本要求后，尝试添加额外功能
   - 修改参数，观察效果变化
   - 组合多个功能

## 常见问题

### Q: 动作没有触发？
检查：
1. 模型文件是否包含该动作组
2. `StartMotion` 的索引是否正确
3. 动作优先级是否足够高

### Q: 表情效果不明显？
检查：
1. 参数值是否足够大（尝试 0.8-1.0）
2. 表情是否每帧都在应用
3. 是否有其他动作覆盖了表情参数

### Q: 拖拽时误触发点击？
检查：
1. 点击阈值是否设置合理（5-10 像素）
2. 点击检测是否在鼠标释放时判断
3. 是否正确重置了拖拽状态
