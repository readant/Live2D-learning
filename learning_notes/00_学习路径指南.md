### 必读文档
1. [Live2D_Py库集成详解](./03_Live2D_Py库集成详解.md) - 掌握 Live2D API
2. [交互系统详解](./06_交互系统详解.md) - 理解交互逻辑

### 核心参数学习

| 参数名 | 作用 | 取值范围 | 练习项目 |
|-------|------|---------|---------|
| ParamAngleX | 头部左右转动 | -30 ~ 30 | 调整头部跟随灵敏度 |
| ParamAngleY | 头部上下转动 | -30 ~ 30 | 添加摇头功能 |
| ParamEyeBallX | 眼球左右移动 | -1 ~ 1 | 调整眼球跟随速度 |
| ParamEyeBallY | 眼球上下移动 | -1 ~ 1 | 自定义眨眼频率 |
| ParamBreath | 呼吸效果 | 0 ~ 1 | 调整呼吸幅度 |

## 实践练习集

### 练习 1：基础参数调试
**目标**：熟悉 Live2D 参数系统

**步骤**：
1. 打开 `src/core/mouse_tracker.py`
2. 找到 `get_gaze_angles()` 方法
3. 修改返回值，观察模型变化
4. 记录每个参数的效果

**扩展**：
- 尝试修改 `MAX_GAZE_ANGLE` 的值
- 调整 `gaze_smoothing` 参数

---

### 练习 2：自定义表情
**目标**：学会添加新表情

**步骤**：
1. 打开 `src/live2d/expression.py`
2. 查看现有的表情配置
3. 添加一个新表情（如"惊讶"）
4. 在 desktop_pet_widget.py 中测试

**模板**：
```python
Expression.SURPRISED: {
    "ParamEyeLOpen": 1.0,    # 睁大眼睛
    "ParamEyeROpen": 1.0,
    "ParamEyeLSmile": 0.0,   # 不笑
    "ParamEyeRSmile": 0.0,
    "ParamMouthForm": 0.0,
    "ParamMouthOpenY": 0.5,  # 张嘴
    "ParamBrowLForm": 0.5,   # 眉毛上扬
    "ParamBrowRForm": 0.5,
    "ParamCheek": 0.0,
},
```

---

### 练习 3：添加新动作
**目标**：理解动作触发机制

**步骤**：
1. 在 `resources/models/Hiyori.model3.json` 中查看现有动作
2. 修改 `src/core/interaction_controller.py` 中的动作配置
3. 添加自定义冷却时间
4. 测试动作触发

**扩展**：
- 实现动作组合播放
- 添加动作完成回调

---

### 练习 4：性能分析
**目标**：学会性能优化

**步骤**：
1. 使用 `time` 模块测量渲染时间
2. 分析 paintGL() 的执行时间
3. 尝试优化：减少参数设置调用
4. 对比优化前后的 FPS

**工具**：
```python
import time

def paintGL(self):
    start = time.time()
    # 渲染代码...
    elapsed = time.time() - start
    if elapsed > 0.033:  # 超过 30ms
        print(f"[性能] 渲染耗时: {elapsed*1000:.2f}ms")
```

## 常见问题自检

在学习过程中遇到问题时，先尝试以下步骤：

### 1. 环境问题
```bash
# 检查 Python 版本
python --version

# 检查依赖安装
pip list | grep -E "live2d|PyQt5"

# 检查模型文件
ls resources/models/
```

### 2. 运行问题
- [ ] 是否激活了虚拟环境？
- [ ] 模型文件路径是否正确？
- [ ] OpenGL 是否正常工作？

### 3. 功能问题
- [ ] 查看控制台输出
- [ ] 启用调试日志
- [ ] 参考 04_常见问题与解决方案.md
