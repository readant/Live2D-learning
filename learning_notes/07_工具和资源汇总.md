## 快速参考

### 常用命令

```bash
# 安装依赖
pip install -r requirements.txt

# 运行程序
python main.py

# 检查 Python 版本
python --version

# 检查已安装的包
pip list

# Git 提交
git add .
git commit -m "描述"
git push origin main
```

### 文件路径参考

```
Live2D-learning/
├── config/settings.py          # 配置文件
├── src/live2d/model.py         # 模型接口
├── src/core/                   # 核心功能
├── src/ui/                     # UI 组件
├── resources/models/           # 模型文件
└── learning_notes/            # 学习笔记
```

### 参数速查

| 参数 | 说明 | 取值范围 |
|------|------|---------|
| ParamAngleX | 头部 X 轴旋转 | -30 ~ 30 |
| ParamAngleY | 头部 Y 轴旋转 | -30 ~ 30 |
| ParamEyeBallX | 眼球 X 位置 | -1 ~ 1 |
| ParamEyeBallY | 眼球 Y 位置 | -1 ~ 1 |
| ParamBreath | 呼吸 | 0 ~ 1 |

### 调试技巧
