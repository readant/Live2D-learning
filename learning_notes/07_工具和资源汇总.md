## 快速参考

### 常用命令

```bash
# 安装依赖（使用国内镜像）
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple

# 运行程序
python main.py

# 检查 Python 版本
python --version

# 检查已安装的包
pip list

# Git 克隆（使用镜像）
git clone https://hub.fastgit.org/EasyLive2D/live2d-py.git

# 创建虚拟环境
python -m venv venv

# 激活虚拟环境（Windows）
venv\Scripts\activate

# 激活虚拟环境（Linux/Mac）
source venv/bin/activate
```

### 文件路径参考

```
Live2D-learning/
├── config/settings.py          # 配置文件
├── src/live2d/model.py         # 模型接口
├── src/core/                   # 核心功能
├── src/ui/                     # UI 组件
├── resources/models/           # 模型文件
└── learning_notes/            # 学习笔记
```
