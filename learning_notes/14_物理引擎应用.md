## 物理引擎概述

### 物理模拟的作用

物理引擎用于模拟头发、衣服、配饰等部位的自然运动：

| 应用部位 | 运动类型 | 示例 |
|----------|----------|------|
| 头发 | 飘动、摇晃 | 行走时头发摆动 |
| 裙子 | 摆动、下垂 | 站立时裙摆 |
| 帽子 | 晃动 | 跳跃时帽子晃动 |
| 尾巴 | 摇摆 | 情绪表达 |
| 配饰 | 晃动 | 耳环、项链摆动 |

### 物理引擎特点

- **实时计算**：根据模型运动状态实时计算形变
- **参数驱动**：通过物理参数控制运动特性
- **碰撞检测**：避免穿透等异常
- **性能优化**：高效的物理计算

### 物理模拟类型

| 类型 | 说明 | 应用 |
|------|------|------|
| 单摆运动 | 简单的来回摆动 | 耳环、项链 |
| 弹簧运动 | 具有弹性的振动 | 帽子、马尾 |
| 阻尼运动 | 逐渐衰减的振动 | 所有物理效果 |
| 风场运动 | 风吹产生的运动 | 头发、裙子 |

## 物理模拟原理

### 弹簧阻尼模型

Live2D 物理引擎基于弹簧阻尼系统：

```
         k: 弹簧刚度
         c: 阻尼系数
         m: 质量

    ●────弹簧(k)────○
    │              │
    │            质量 m
    │              │
    └──阻尼器(c)───┘
```

### 运动方程

```python
# 弹簧阻尼运动方程
# m * a + c * v + k * x = F

def physics_update(mass, stiffness, damping, force, position, velocity, dt):
    # 加速度
    acceleration = (force - stiffness * position - damping * velocity) / mass

    # 速度更新
    new_velocity = velocity + acceleration * dt

    # 位置更新
    new_position = position + new_velocity * dt

    return new_position, new_velocity
```

### 常用参数

| 参数 | 说明 | 典型值 |
|------|------|--------|
| 刚度 (Stiffness) | 弹性强度 | 0.1 ~ 0.5 |
| 阻尼 (Damping) | 衰减速度 | 0.3 ~ 0.8 |
| 质量 (Mass) | 运动惯性 | 1.0 ~ 3.0 |
| 重力 (Gravity) | 下沉力量 | 0.0 ~ 1.0 |

## 项目中的物理应用

### 鼠标追踪物理

项目中的视线追踪使用平滑跟随：

```python
class MouseTracker:
    def __init__(self):
        self.current_x = 0.0
        self.current_y = 0.0
        self.target_x = 0.0
        self.target_y = 0.0
        self.smoothing = 0.15  # 平滑系数

    def update(self, mouse_x, mouse_y):
        # 设置目标位置
        self.target_x = mouse_x
        self.target_y = mouse_y

        # 平滑插值
        self.current_x += (self.target_x - self.current_x) * self.smoothing
        self.current_y += (self.target_y - self.current_y) * self.smoothing

    def get_gaze_angles(self):
        # 转换为角度
        angle_x = self.current_x * 30.0  # -30 ~ 30
        angle_y = self.current_y * 30.0
        return angle_x, angle_y
```

### 呼吸物理效果

呼吸是一种周期性的物理运动：

```python
class BreathPhysics:
    def __init__(self):
        self.phase = 0.0
        self.speed = 0.8  # 呼吸速度
        self.base_amplitude = 0.02
        self.current_value = 0.0

    def update(self, delta_time):
        # 更新相位
        self.phase += delta_time * self.speed
        if self.phase > 2 * math.pi:
            self.phase -= 2 * math.pi

        # 计算呼吸值（正弦波）
        self.current_value = math.sin(self.phase) * self.base_amplitude

        return self.current_value

    def apply_to_model(self, model):
        value = self.update(0.016)  # 假设16ms帧
        model.set_parameter("ParamBreath", value)
```

### 头发飘动效果

```python
class HairPhysics:
    def __init__(self):
        self.angle = 0.0
        self.velocity = 0.0
        self.stiffness = 0.3  # 刚度
        self.damping = 0.5    # 阻尼

    def update(self, body_angle, delta_time):
        # 身体运动产生的力
        force = body_angle * 0.1

        # 弹簧阻尼计算
        acceleration = (force - self.stiffness * self.angle - self.damping * self.velocity) / 1.0
        self.velocity += acceleration * delta_time
        self.angle += self.velocity * delta_time

        return self.angle

    def apply_to_model(self, model):
        hair_angle = self.update(self.body_angle, 0.016)
        model.set_parameter("ParamHairBack", hair_angle)
```

## 自定义物理效果

### 弹性效果

模拟弹性物体的运动：

```python
class SpringPhysics:
    def __init__(self, stiffness=0.2, damping=0.5):
        self.value = 0.0
        self.target = 0.0
        self.velocity = 0.0
        self.stiffness = stiffness
        self.damping = damping

    def set_target(self, target):
        self.target = target

    def update(self, delta_time):
        # 弹簧力
        spring_force = (self.target - self.value) * self.stiffness
        # 阻尼力
        damping_force = -self.velocity * self.damping
        # 加速度
        acceleration = spring_force + damping_force

        # 更新速度和位置
        self.velocity += acceleration * delta_time
        self.value += self.velocity * delta_time

        return self.value
```

### 重力效果

模拟下垂类的重力效果：

```python
class GravityPhysics:
    def __init__(self, gravity=0.3, max_angle=45.0):
        self.gravity = gravity
        self.max_angle = max_angle
        self.current_angle = 0.0
        self.velocity = 0.0

    def update(self, body_tilt, delta_time):
        # 重力影响
        gravity_force = self.gravity * math.sin(math.radians(body_tilt))

        # 角度限制
        target = gravity_force * self.max_angle

        # 弹性跟随
        diff = target - self.current_angle
        self.velocity += diff * 0.1
        self.velocity *= 0.95  # 阻尼
        self.current_angle += self.velocity

        return self.current_angle
```

### 风力效果

模拟风吹效果：

```python
class WindPhysics:
    def __init__(self):
        self.base_wind = 0.0
        self.wind_strength = 0.0
        self.wind_phase = 0.0
        self.affected_hair = []

    def set_wind(self, direction, strength):
        """设置风向和强度

        Args:
            direction: 风向角度（度）
            strength: 风力强度（0.0 ~ 1.0）
        """
        self.base_wind = direction
        self.wind_strength = strength

    def update(self, delta_time):
        # 更新风相位（模拟阵风）
        self.wind_phase += delta_time * 2.0
        gust = math.sin(self.wind_phase) * 0.3 + 0.7  # 0.4 ~ 1.0

        # 计算有效风力
        effective_wind = self.wind_strength * gust

        # 应用到各部位
        results = {}
        for hair_id, sensitivity in self.affected_hair:
            wind_angle = self.base_wind + math.sin(self.wind_phase) * 10
            results[hair_id] = wind_angle * sensitivity * effective_wind

        return results
```

## 常见问题

### Q1: 物理运动不自然

**原因**：参数配置不当

**解决**：
- 调整刚度和阻尼系数
- 检查输入源权重
- 确认输出缩放合适

### Q2: 运动响应延迟

**原因**：阻尼过大或刚度过小

**解决**：
- 降低阻尼系数
- 提高刚度
- 优化更新频率

### Q3: 运动幅度过大

**原因**：输出缩放过大

**解决**：
- 减小 Scale 参数
- 调整 Normalization 范围
- 降低输入权重
