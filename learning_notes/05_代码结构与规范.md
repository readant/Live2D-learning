## 注释规范

### 注释原则

1. **不要注释显而易见的代码**
2. **注释为什么，而不是做什么**
3. **保持注释更新**

```python
# 不好：注释显而易见的事实
x = x + 1  # x 加 1

# 好：注释原因
x = x + 1  # 跳过第一个动作，因为它是过渡动画
```

### 行内注释

```python
# 使用空格分隔注释和代码
x = x + 1  # 跳过第一个动作

# 避免
x=x+1#x加1
```

### 私有命名

```python
class Model:
    def __init__(self):
        self.public_attr = 1      # 公开属性
        self._protected_attr = 2  # 受保护属性（约定）
        self.__private_attr = 3   # 私有属性（名称改写）
```

## PyQt5 特定规范

### 信号和槽

```python
from PyQt5.QtCore import pyqtSignal, QObject

class MyWidget(QObject):
    # 定义信号
    value_changed = pyqtSignal(int)
    expression_changed = pyqtSignal(str)
    
    def __init__(self):
        super().__init__()
        # 连接信号
        self.value_changed.connect(self.on_value_changed)
    
    def emit_signal(self, value: int):
        self.value_changed.emit(value)
    
    def on_value_changed(self, value: int):
        print(f"值已改变: {value}")
```

### Widget 层级

```python
# 好：清晰的初始化顺序
def __init__(self):
    super().__init__()
    self._init_ui()      # 1. UI 初始化
    self._init_model()   # 2. 模型初始化

def _init_ui(self):

def _init_model(self):

    """初始化定时器"""
    self.timer = QTimer(self)
```

## 性能注意事项

### 避免在循环中创建对象

```python
# 不好
def paintEvent(self, event):
    for i in range(100):
        pen = QPen()  # 每次都创建新对象
        painter.setPen(pen)

# 好
def __init__(self):
    self._pen = QPen()  # 在初始化时创建一次

def paintEvent(self, event):
    for i in range(100):
        painter.setPen(self._pen)
```

### 使用局部变量

```python
# 不好
def update(self):
    self.x = self.x + 1
    self.y = self.y + 1
    self.z = self.z + 1

# 好
def update(self):
    x, y, z = self.x, self.y, self.z
    x = x + 1
    y = y + 1
    z = z + 1
    self.x, self.y, self.z = x, y, z
```

### 合理使用 `__slots__`

```python
# 可以减少内存占用
class Point:
    __slots__ = ['x', 'y']
    
    def __init__(self, x, y):
        self.x = x
        self.y = y
```

## 测试建议

### 单元测试基础

```python
# tests/test_mouse_tracker.py
import unittest
from src.core.mouse_tracker import MouseTracker

class TestMouseTracker(unittest.TestCase):
    def setUp(self):
        self.tracker = MouseTracker((100, 100))
    
    def test_initial_position(self):
        """测试初始位置"""
        angles = self.tracker.get_gaze_angles()
        self.assertEqual(angles, (0, 0, 0, 0))
```
