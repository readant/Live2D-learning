## 🔬 原理揭秘系列

### 核心技术原理

| 主题 | 原理揭秘 | 相关模块 |
|------|----------|----------|
| **人脸检测** | MediaPipe Face Mesh 深度学习模型原理 | stage1_mediapipe.py |
| **姿态估计** | PnP 算法、3D-2D 投影原理 | stage3_head_pose.py |
| **平滑滤波** | EWMA、卡尔曼滤波数学原理 | stage5_smoothing.py |
| **参数映射** | 线性插值、曲线映射原理 | stage4_parameter_mapping.py |
| **物理引擎** | 弹簧-阻尼系统原理 | 14_物理引擎应用.md |

### Live2D 核心概念

| 概念 | 说明 |
|------|------|
| **MOC3** | Live2D 模型文件格式，存储网格数据 |
| **Model3.json** | 模型配置文件，定义参数和资源 |
| **Motion3.json** | 动作数据文件，关键帧动画 |
| **Physics3.json** | 物理模拟配置 |
| **Expression** | 表情配置，参数组合 |

## 🎯 应用场景发散

Live2D 技术具有广泛的应用场景，掌握核心技术后可探索以下方向：

| 领域 | 应用场景 | 说明 |
|------|----------|------|
| 🎮 **游戏领域** | 角色立绘、虚拟 NPC、表情系统 | AVG 游戏、互动游戏 |
| 🎤 **虚拟主播** | 实时直播、短视频创作、直播互动 | 面部捕捉驱动 |
