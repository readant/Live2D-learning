## 国内镜像源配置

### PyPI 镜像

由于国外 PyPI 源在国内访问较慢，建议配置国内镜像源。

#### 临时使用

每次安装包时指定镜像：

```bash
# 清华镜像（推荐）
pip install package_name -i https://pypi.tuna.tsinghua.edu.cn/simple

# 阿里镜像
pip install package_name -i https://mirrors.aliyun.com/pypi/simple/

# 中科大镜像
pip install package_name -i https://pypi.mirrors.ustc.edu.cn/simple/

# 豆瓣镜像
pip install package_name -i https://pypi.douban.com/simple/
```

#### 永久配置（推荐）

**Windows 用户：**

在用户目录下创建 `pip` 文件夹，新建文件 `pip.ini`：

```ini
[global]
index-url = https://pypi.tuna.tsinghua.edu.cn/simple
timeout = 60
```

路径示例：`C:\Users\你的用户名\pip\pip.ini`

**Linux/Mac 用户：**

在用户目录下创建 `.pip` 文件夹，新建文件 `pip.conf`：

```bash
mkdir ~/.pip
cat > ~/.pip/pip.conf << EOF
[global]
index-url = https://pypi.tuna.tsinghua.edu.cn/simple
timeout = 60
EOF
```

### Anaconda 镜像

如果使用 Anaconda，配置国内镜像：

```bash
# 添加清华 Anaconda 镜像
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main/
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/conda-forge/
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/msys2/
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/bioconda/
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/menpo/

# 设置搜索时显示通道地址
conda config --set show_channel_urls yes
```

### Git 镜像

克隆 GitHub 仓库较慢时，可以使用国内镜像：

#### GitHub 镜像网站

- **Gitee**：https://gitee.com/
- **GitHub 镜像**：https://hub.fastgit.org/

#### 使用方法

```bash
# 克隆仓库（使用 fastgit 镜像）
git clone https://hub.fastgit.org/EasyLive2D/live2d-py.git

# 添加远程仓库
git remote add origin https://hub.fastgit.org/EasyLive2D/live2d-py.git

# 拉取代码
git pull origin main
```

#### Gitee 镜像

许多项目在 Gitee 上有镜像：

```bash
# 搜索 Gitee 上的项目
# 例如：https://gitee.com/mirrors/live2d-py
git clone https://gitee.com/mirrors/live2d-py.git
```

## 开发工具安装

### Python 安装

#### Windows 系统

1. 访问 [Python 官网](https://www.python.org/downloads/windows/)
2. 下载最新稳定版本（推荐 3.10+）
3. 安装时勾选 "Add Python to PATH"
4. 验证安装：

```bash
python --version
pip --version
```

#### 国内 Python 下载源

如果官网下载慢，可以使用国内镜像：

- **清华镜像**：https://mirrors.tuna.tsinghua.edu.cn/python/
- **华为镜像**：https://repo.huaweicloud.com/python/

### IDE 安装

#### VS Code（推荐）

1. 访问 [VS Code 官网](https://code.visualstudio.com/)
2. 下载对应系统版本
3. 安装中文插件：搜索 "Chinese" 安装中文语言包
4. 安装 Python 插件：搜索 "Python" 安装

#### PyCharm

1. 访问 [PyCharm 官网](https://www.jetbrains.com/pycharm/)
2. 下载社区版（免费）或专业版
3. 安装完成后配置 Python 解释器

### Git 安装

#### Windows 系统

1. 访问 [Git 官网](https://git-scm.com/download/win)
2. 下载并安装
3. 验证安装：

```bash
git --version
```

#### 国内 Git 下载

- **淘宝镜像**：https://npm.taobao.org/mirrors/git-for-windows/

## 环境变量配置

### Windows 系统

1. 右键「此电脑」→「属性」→「高级系统设置」→「环境变量」
2. 在「用户变量」中添加：

| 变量名 | 值 | 说明 |
|--------|------|------|
| PYTHONPATH | `你的项目目录` | Python 模块搜索路径 |
| PATH | 添加 Python 和 pip 路径 | 命令行可执行 |

### 配置示例

```bash
# 设置 Python 路径（Windows 命令行）
setx PYTHONPATH "E:\Projects0\Trae\Live2D-learning"

# 设置环境变量（永久生效）
setx PATH "%PATH%;C:\Python310;C:\Python310\Scripts"
```

## 项目环境配置

### 推荐步骤

```bash
# 1. 克隆项目（使用镜像）
git clone https://hub.fastgit.org/你的用户名/Live2D-learning.git
cd Live2D-learning

# 2. 创建虚拟环境（推荐）
python -m venv venv

# 3. 激活虚拟环境
# Windows
venv\Scripts\activate
# Linux/Mac
source venv/bin/activate

# 4. 使用国内镜像安装依赖
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple

# 5. 运行项目
python main.py
```

### 虚拟环境使用指南

```bash
# 创建虚拟环境
python -m venv venv

# 激活虚拟环境
# Windows
venv\Scripts\activate
# Linux/Mac
source venv/bin/activate

# 退出虚拟环境
deactivate

# 安装包到虚拟环境
pip install package_name

# 导出依赖清单
pip freeze > requirements.txt
```

## 常见问题

### 1. pip 安装失败

**问题**：连接超时或找不到包

**解决方案**：

```bash
# 使用国内镜像
pip install package_name -i https://pypi.tuna.tsinghua.edu.cn/simple

# 升级 pip
python -m pip install --upgrade pip -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 2. Git 克隆失败

**问题**：无法连接到 GitHub

**解决方案**：

```bash
# 使用镜像地址
git clone https://hub.fastgit.org/EasyLive2D/live2d-py.git

# 或使用 Gitee 镜像
git clone https://gitee.com/mirrors/live2d-py.git
```

### 3. Python 版本问题

**问题**：提示 Python 版本过低

**解决方案**：

- 安装 Python 3.10+ 版本
- 使用 pyenv 管理多个 Python 版本

### 4. 权限问题

**问题**：Windows 下安装包提示权限不足

**解决方案**：

- 以管理员身份运行命令行
- 或使用 `--user` 参数：

```bash
pip install package_name --user
```

### 5. Live2D 模型加载失败

**问题**：模型文件路径错误或缺失

**解决方案**：

```bash
# 检查模型文件是否存在
dir resources/models/

# 确认配置文件中的路径正确
```
