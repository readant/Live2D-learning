## 阶段一：MediaPipe 人脸检测

### 核心概念

**MediaPipe Face Mesh** 是 Google 开发的实时人脸关键点检测模型，可检测 468 个 3D 关键点。

### 关键代码

```python
class FaceMeshDetector:
    def __init__(self, static_image_mode=False, max_num_faces=1):
        self.mp_face_mesh = mp.solutions.face_mesh
        self.face_mesh = self.mp_face_mesh.FaceMesh(
            static_image_mode=static_image_mode,
            max_num_faces=max_num_faces,
            refine_landmarks=True,  # 启用精确关键点
            min_detection_confidence=0.5,
            min_tracking_confidence=0.5
        )
    
    def process_frame(self, frame):
        """处理单帧图像，返回检测结果"""
        rgb_frame = cv2.cvtColor(frame, cv2.COLOR_BGR2RGB)
        rgb_frame.flags.writeable = False  # 提高性能
        return self.face_mesh.process(rgb_frame)
```

### 原理揭秘

**Face Mesh 工作流程**：
1. **人脸检测**：使用 BlazeFace 检测人脸位置
2. **关键点回归**：使用 ResNet18 预测 468 个关键点坐标
3. **3D 重建**：基于关键点进行深度估计

**468 点分布**：
- 下巴轮廓：0-16
- 左眼：36-45
- 右眼：42-51
- 嘴巴：61-82
- 眉毛：17-26（左）、27-35（右）

## 阶段二：面部特征点提取

### 核心概念

从 468 个关键点中提取有意义的面部特征：
- 眼睛开合度
- 嘴巴开合度
- 微笑程度
- 眉毛高度

### 关键代码

```python
class FacialLandmarks:
    # 关键点分组定义
    LEFT_EYE = list(range(36, 42))
    RIGHT_EYE = list(range(42, 48))
    MOUTH_OUTER = list(range(48, 61))
    MOUTH_INNER = list(range(60, 68))
    LEFT_BROW = list(range(17, 22))
    RIGHT_BROW = list(range(22, 27))
    
    def extract_eye_features(self, landmarks):
        """提取眼睛特征"""
        left_eye = landmarks[self.LEFT_EYE]
        right_eye = landmarks[self.RIGHT_EYE]
        
        # 计算眼睛高度与宽度的比值（开合度）
        left_openness = self._calculate_eye_openness(left_eye)
        right_openness = self._calculate_eye_openness(right_eye)
        
        return {
            'left_openness': left_openness,
            'right_openness': right_openness,
            'avg_openness': (left_openness + right_openness) / 2
        }
```

### 原理揭秘

**眼睛开合度计算**：
- 垂直距离：上下眼睑关键点的距离
- 水平距离：眼角之间的距离
- 开合度 = 垂直距离 / 水平距离

**微笑检测**：
- 测量嘴角上扬角度
- 计算嘴巴宽度与高度的比值

## 阶段三：头部姿态估计

### 核心概念

**PnP（Perspective-n-Point）算法**：从 2D 图像点估计 3D 物体姿态。

### 关键代码

```python
class HeadPoseEstimator:
    def __init__(self):
        # 3D 人脸模型关键点（基于平均人脸模型）
        self.model_points = np.array([
            (0.0, 0.0, 0.0),           # 鼻尖
            (0.0, -330.0, -65.0),      # 下巴
            (-225.0, 170.0, -135.0),   # 左眼外角
            (225.0, 170.0, -135.0),    # 右眼外角
            (-150.0, -150.0, -125.0),  # 左嘴角
            (150.0, -150.0, -125.0)    # 右嘴角
        ], dtype=np.float32)
        
        # 相机内参（近似值）
        self.camera_matrix = np.array([
            [640, 0, 320],
            [0, 640, 240],
            [0, 0, 1]
        ], dtype=np.float32)
    
    def estimate(self, landmarks_2d, frame_shape):
        """估计头部姿态"""
        image_points = self._get_image_points(landmarks_2d)
        
        success, rotation_vec, translation_vec = cv2.solvePnP(
            self.model_points,
            image_points,
            self.camera_matrix,
            None,
            flags=cv2.SOLVEPNP_ITERATIVE
        )
        
        # 旋转向量转欧拉角
        rotation_matrix, _ = cv2.Rodrigues(rotation_vec)
        euler_angles = self._rotation_matrix_to_euler(rotation_matrix)
        
        return {
            'rotation_vector': rotation_vec,
            'euler_angles': euler_angles,
            'success': success
        }
```

### 原理揭秘

**PnP 算法原理**：
1. 已知 3D 模型点坐标（人脸关键点的平均 3D 位置）
2. 检测到 2D 图像点坐标
3. 通过最小化重投影误差估计相机姿态

**欧拉角定义**：
- **Pitch**（俯仰）：上下点头
- **Yaw**（偏航）：左右转头
- **Roll**（翻滚）：头部倾斜

## 阶段四：参数映射

### 核心概念

将面部捕捉数据映射到 Live2D 参数，包括：
- 角度参数：ParamAngleX/Y/Z
- 眼睛参数：ParamEyeBallX/Y, ParamEyeLid{Left/Right}
- 嘴巴参数：ParamMouthOpenY, ParamMouthForm
- 眉毛参数：ParamEyebrow{Left/Right}

### 关键代码

```python
class ParameterMapper:
    def __init__(self):
        # 参数映射配置
        self.mappings = {
            'pitch': {'param': 'ParamAngleX', 'range': (-30, 30)},
            'yaw': {'param': 'ParamAngleY', 'range': (-30, 30)},
            'roll': {'param': 'ParamAngleZ', 'range': (-30, 30)},
            'eye_open': {'param': 'ParamEyeLidLeft', 'range': (0, 1)},
            'mouth_open': {'param': 'ParamMouthOpenY', 'range': (0, 1)},
            'smile': {'param': 'ParamMouthForm', 'range': (-1, 1)}
        }
    
    def map_pose_to_parameters(self, pose):
        """将姿态数据映射到参数"""
        parameters = {}
        if pose and pose.get('success'):
            euler_angles = pose['euler_angles']
            parameters['ParamAngleX'] = self._normalize(euler_angles[0], -np.pi/2, np.pi/2, -1, 1)
            parameters['ParamAngleY'] = self._normalize(euler_angles[1], -np.pi/2, np.pi/2, -1, 1)
            parameters['ParamAngleZ'] = self._normalize(euler_angles[2], -np.pi/2, np.pi/2, -1, 1)
        return parameters
```

### 原理揭秘

**参数映射流程**：
1. **输入裁剪**：将输入值限制在合理范围
2. **归一化**：映射到 [0, 1] 区间
3. **曲线应用**：应用非线性曲线（如 ease_in_out）
4. **输出映射**：映射到 Live2D 参数范围

## 阶段五：平滑滤波

### 核心概念

**抖动抑制**是面部捕捉的关键，常用算法：
- **Moving Average**：移动平均滤波
- **EWMA**：指数加权移动平均
- **Kalman Filter**：卡尔曼滤波

### 关键代码

```python
class MultiStageSmoother:
    def __init__(self):
        self.outlier_threshold = 0.3
        self.ewma_alpha = 0.2  # 平滑系数
        self.kalman_smoother = None
        self.previous_parameters = {}
    
    def smooth(self, parameters):
        """多级平滑处理"""
        if not self.previous_parameters:
            self.previous_parameters = parameters
            return parameters
        
        # 1. 异常值过滤
        parameters = self._outlier_filter(parameters)
        
        # 2. EWMA 滤波
        parameters = self._ewma_filter(parameters)
        
        # 3. 速度限制
        parameters = self._velocity_limit(parameters)
        
        self.previous_parameters = parameters
        return parameters
    
    def _ewma_filter(self, parameters):
        """指数加权移动平均"""
        smoothed = {}
        for key, value in parameters.items():
            prev_value = self.previous_parameters.get(key, value)
            smoothed[key] = self.ewma_alpha * value + (1 - self.ewma_alpha) * prev_value
        return smoothed
```

### 原理揭秘

**EWMA 公式**：
```
S_t = α * X_t + (1 - α) * S_{t-1}
```
- α：平滑系数（0-1），越小越平滑
- X_t：当前值
- S_{t-1}：上一次平滑值

**卡尔曼滤波**：
- 预测：基于运动模型预测下一状态
- 更新：结合测量值修正预测

## 阶段六：系统集成

### 核心概念

将所有模块整合为完整的面部捕捉系统。

### 关键代码

```python
class FaceCaptureSystem:
    def __init__(self, smoothing_enabled=True):
        self.detector = FaceMeshDetector()
        self.feature_extractor = FacialLandmarks()
        self.pose_estimator = HeadPoseEstimator()
        self.parameter_mapper = ParameterMapper()
        self.smoother = MultiStageSmoother()
        self.smoothing_enabled = smoothing_enabled
    
    def process_frame(self, frame):
        """处理单帧图像，返回参数"""
        # 水平翻转（镜像效果）
        frame = cv2.flip(frame, 1)
        
        # 1. 人脸检测
        results = self.detector.process_frame(frame)
        
        if not results.multi_face_landmarks:
            return frame, None
        
        # 2. 获取关键点数组
        landmarks = self.detector.get_landmarks_array(results)
        
        # 3. 提取表情特征
        features = self.feature_extractor.extract_all_features(landmarks)
        
        # 4. 姿态估计
        pose = self.pose_estimator.estimate(landmarks, frame.shape)
        
        # 5. 参数映射
        parameters = self.parameter_mapper.map_all_features(
            pose, 
            features.get('eyes'), 
            features.get('mouth'),
            features.get('eyebrows')
        )
        
        # 6. 平滑处理
        if self.smoothing_enabled and parameters:
            parameters = self.smoother.smooth(parameters)
        
        return frame, parameters
```

### 原理揭秘

**数据流管理**：
- 每帧处理时间应控制在 16ms 以内（60fps）
- 使用缓冲队列处理异步数据
- 状态管理确保数据一致性

## 阶段七：动作录制

### 核心概念

录制面部捕捉数据并导出为 Live2D motion3.json 格式。

### 关键代码

```python
class MotionRecorder:
    def __init__(self, fps=30):
        self.fps = fps
        self.is_recording = False
        self.recorded_frames = []
        self.start_time = None
        
        # motion3.json 元数据
        self.meta = {
            'Title': 'Recorded Motion',
            'Duration': 0,
            'Fps': fps,
            'Loop': True
        }
    
    def record_frame(self, parameters):
        """记录单帧参数"""
        if self.is_recording and parameters:
            self.recorded_frames.append(parameters.copy())
    
    def export_to_motion3(self, filepath):
        """导出为 motion3.json"""
        duration = len(self.recorded_frames) / self.fps
        
        # 构建曲线数据
        curves = []
        for param_name in self.recorded_frames[0].keys():
            curve_data = {
                'Target': param_name,
                'Id': 0,
                'Type': 'Float',
                'Fps': self.fps,
                'TotalFrames': len(self.recorded_frames),
                'Segments': []
            }
            
            # 简化：线性插值的单段曲线
            values = [frame[param_name] for frame in self.recorded_frames]
            curve_data['Segments'].append({
                'Frame': 0,
                'Value': values[0],
                'Easing': 'Linear',
                'EndFrame': len(self.recorded_frames) - 1,
                'EndValue': values[-1]
            })
            
            curves.append(curve_data)
        
        motion_data = {
            'Version': '3.0',
            'Meta': self.meta,
            'Duration': int(duration * 1000),
            'Fps': self.fps,
            'Curves': curves,
            'UserData': []
        }
        
        with open(filepath, 'w', encoding='utf-8') as f:
            json.dump(motion_data, f, indent=2, ensure_ascii=False)
```

### 原理揭秘

**motion3.json 格式**：
- **Version**：格式版本
- **Meta**：元数据（标题、时长等）
- **Curves**：参数曲线数组
- **Segments**：曲线段（支持不同插值方式）
